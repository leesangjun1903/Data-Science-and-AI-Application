# Mapper : Topological Methods for the Analysis of High Dimensional Data Sets and 3D Object Recognition

# 핵심 요약 및 기여

**주요 주장:**  
이 논문은 고차원 데이터 세트와 3D 객체의 구조적 특성을 보존하면서 단순화된 이산적인 표현을 추출하는 새로운 토폴로지 기반 방법인 **Mapper**를 제안한다. Mapper는 실수값 필터 함수(filter function)에 따라 데이터 클러스터의 상호교차 구조를 그래프나 심플렉스 복합체(simplicial complex)로 변환함으로써 대규모·고차원 데이터의 형상(shape)과 위상학적(topological) 정보를 저차원 구조에 압축하여 시각화하고 분석할 수 있게 한다.[1]

**주요 기여:**  
- **일반화된 Reeb 그래프**: 연속적인 필터 함수에 대해 다중 해상도(multi-resolution)의 그래프 및 고차원 심플렉스 복합체를 생성하며, 이는 전통적 Reeb 그래프(1차원)에서 벗어나 임의 차원의 구조를 표현한다.  
- **필터 함수와 군집화 분리**: 필터 함수 설계와 군집화 알고리즘을 독립적으로 적용할 수 있어, 응용 분야별·도메인별 최적화가 가능하다.  
- **다중 해상도 검증**: 필터 구간 크기와 겹침(overlap)을 변화시키며 생성되는 복합체 간 맵핑을 통해 위상학적 특징의 지속성(persistence)을 평가할 수 있어, 노이즈와 아티팩트 분리에 유리하다.  
- **3D 객체 인식 응용**: 3D 메쉬 데이터에도 동일한 절차를 적용하여 객체의 스켈레톤 구조 추출, 포즈 불변 인식(shape matching) 등의 다양한 응용 가능성을 보인다.

# 문제 정의 및 제안 방법

## 1. 해결 과제  
- 고차원(예: 수백 차원)의 포인트 클라우드(point cloud)나 수많은 3D 메쉬 표본에 대해 대규모·저차원 투사 투시(projection pursuit)만으로는 감지하기 어려운 위상 구조(루프, 빈 공간, 분기점 등)를 안정적으로 추출해야 한다.

## 2. 제안 방법: Mapper  
1. **필터 함수 선택**: 데이터 점 $$X\subset\mathbb{R}^D$$에 대해  

$$
     f: X \to Z
   $$  
   
   로 매핑하는 하나 이상의 연속 함수(밀도, 편심도(eccentricity), 그래프 라플라시안 고유벡터 등)를 정의한다.  
2. **매개변수 공간 분할**: $$Z$$의 범위를 중첩(interleaving) 구간(1차원) 또는 셀 복셀(고차원)으로 덮개(covering)해 해상도 $$l$$, 겹침 비율 $$p$$을 설정한다.  
3. **부분 군집화(Partial Clustering)**: 각 구간에 대응하는 데이터 부분집합 $$f^{-1}(I)$$에서 표준 군집화 알고리즘(예: single-linkage)을 적용해 클러스터 $$\{C_{j,k}\}$$를 생성한다.  
4. **교차 연결**: 서로 다른 구간의 클러스터 간 겹침이 있을 때 그래프 간선 또는 고차원 심플렉스(삼각형, 사면체 등)를 생성해 nerve 복합체를 구성한다.  
5. **다중 해상도 체인**: 해상도를 변화시키며 얻은 일련의 복합체들 사이에 단사 사상(simplicial map)을 정의해, 위상 특징의 지속성(persistence)을 평가한다.

수식 예시:  
- 편심도 함수 $$E_p(x)=\bigl(\tfrac{1}{|X|}\sum_{y\in X}d(x,y)^p\bigr)^{1/p}$$  
- 그래프 라플라시안 고유벡터 $$\phi_i$$를 필터 함수로 사용

## 3. 모델 구조  
- 결과물은 **1-스켈레톤(graph)** 또는 **k-스켈레톤(고차원 심플렉스 복합체)** 형태  
- 각 노드(vertex)는 클러스터, 엣지(edge)는 클러스터 간 교차, 높은 차원 심플렉스는 다중 클러스터 교차로 구성  
- 필터 함수, 구간 해상도, 군집화 알고리즘이 유연하게 조합 가능

# 성능 향상 및 한계

**성능 향상**  
- 데이터 크기를 획기적으로 축소하면서도 위상 특징(연결성, 루프, 공동체 구조)을 보존  
- 고차원 투사 기반 방법에 비해 민감도(metric choice) 완화 및 노이즈에 강건  
- 3D Shape Database 실험에서 포즈 변형에도 스켈레톤 유사성 유지 및 포즈 불변 매칭 오류율 3.03% 달성  

**한계**  
- **군집화 스케일 선택**: 히스토그램 휴리스틱이 클러스터 밀도 차에 민감하며, 이상 분포 시 부정확 가능  
- **매개변수 조정 필요**: 해상도($$l,p$$)와 필터 함수 선택에 따라 결과가 크게 달라짐  
- **계산 복잡도**: 고차원 매개변수 공간과 대규모 데이터에서 군집화·교차 검사 비용이 증가

# 일반화 성능 향상 가능성

- **지속성 기반 필터링**: 멀티해상도 체인에서 진짜 위상 특징만 선택해 노이즈 영향 최소화  
- **적응형 군집화**: 지역 밀도 기반으로 해상도와 스케일을 자동 조정하는 기법 도입  
- **다중 필터 결합**: 서로 보완적인 정보(라플라시안 고유벡터, 편심도, 밀도)를 통합해 모델의 표현력 강화

# 향후 연구 영향 및 고려사항

**연구 영향**  
- **토폴로지 기반 데이터 분석(TDA)**의 범용 도구로 자리매김, 생명과학·소셜 네트워크·의료 영상 등 다양한 분야 응용  
- **3D 객체 인식**에서 스켈레톤 추출·포즈 불변 매칭·형상 군집화 등 핵심 모듈로 사용  
- **해석가능 AI** 분야에서 복합체 구조를 해석도로 활용 가능

**고려사항**  
- 대규모 스트리밍 데이터에 실시간 적용을 위한 알고리즘 경량화  
- 매개변수 자동 튜닝(해상도·겹침·군집화 파라미터) 프레임워크 통합  
- 복합체 위상 정보를 downstream task(분류·회귀·클러스터링)에 효과적으로 임베딩하는 방법론 연구
