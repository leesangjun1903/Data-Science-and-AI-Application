# RISE: Randomized Input Sampling for Explanation of Black-box Models 

## 핵심 주장과 주요 기여

**RISE**는 Boston University 연구팀이 제안한 **블랙박스 딥러닝 모델 설명 방법**으로, 모델 내부 구조에 접근하지 않고도 픽셀 수준의 중요도 맵을 생성할 수 있는 혁신적 접근법입니다[1]. 기존 방법들이 그래디언트나 특성맵 등 모델 내부 정보에 의존했던 것과 달리, RISE는 순수하게 입출력 관계만을 활용하여 설명을 생성합니다[1].

## 해결하고자 하는 문제

RISE가 해결하고자 하는 주요 문제들은 다음과 같습니다[1]:

1. **모델 내부 의존성**: 기존 설명 방법들이 그래디언트, 가중치, 특성맵 등 모델 내부 구조에 의존
2. **아키텍처 제한성**: 특정 네트워크 아키텍처에만 적용 가능한 한계
3. **인과관계 평가 부족**: 설명의 실제 인과관계를 평가할 객관적 메트릭 부재
4. **범용성 부족**: 다양한 블랙박스 모델에 일반적으로 적용할 수 있는 방법 부재

## 제안하는 방법론 및 수식

### 핵심 수학적 정의

RISE의 핵심 아이디어는 픽셀의 중요도를 조건부 기댓값으로 정의하는 것입니다[1]:

**픽셀 중요도 정의**:
$$ S_{I,f}(λ) = E_M[f(I ⊙ M) | M(λ) = 1] $$

여기서:
- $$I$$: 입력 이미지
- $$f$$: 블랙박스 모델
- $$M$$: 랜덤 바이너리 마스크
- $$λ$$: 픽셀 위치
- $$⊙$$: 원소별 곱셈

### Monte Carlo 근사

실제 구현에서는 Monte Carlo 샘플링을 통해 근사합니다[1]:

$$ S_{I,f}(λ) ≈ \frac{1}{E[M] \cdot N} \sum_{i=1}^{N} f(I ⊙ M_i) \cdot M_i(λ) $$

### 마스크 생성 과정

1. **초기 마스크 생성**: $$h×w$$ 크기의 이진 마스크를 확률 $$p$$로 생성
2. **업샘플링**: bilinear interpolation으로 $$H×W$$ 크기로 확대
3. **랜덤 시프트**: 공간적으로 랜덤하게 이동하여 다양성 확보
4. **모델 프로빙**: 마스크된 이미지들로 모델의 출력 확률 획득
5. **가중 평균**: 마스크들을 출력 확률로 가중평균하여 최종 중요도 맵 생성[1]

## 모델 구조

RISE의 구조는 다음과 같이 구성됩니다[1]:

1. **입력 계층**: 원본 이미지 $$I$$
2. **마스크 생성기**: 랜덤 바이너리 마스크 $$M_i$$ (i=1,...,N) 생성
3. **블랙박스 모델**: 마스크된 이미지 $$I⊙M_i$$를 입력받아 확률값 출력
4. **중요도 계산기**: 마스크들과 확률값들의 가중합으로 픽셀별 중요도 맵 $$S_{I,f}$$ 생성

## 성능 향상 결과

### 정량적 성능 개선

**ImageNet 데이터셋**에서의 성능 비교[1]:
- **ResNet50**: Deletion AUC 0.1076 (GradCAM 0.1232 대비 12.7% 향상)
- **VGG16**: Deletion AUC 0.0980 (GradCAM 0.1087 대비 9.8% 향상)

**PASCAL VOC 데이터셋**에서의 Pointing Game 정확도[1]:
- **VGG16**: 87.33% (기존 최고 80.00% 대비 7.33%p 향상)

**MS-COCO 데이터셋**에서의 성능[1]:
- **VGG16**: Pointing accuracy 50.71% (기존 최고 49.60% 대비 향상)

### 주요 장점

1. **우수한 인과적 설명**: deletion/insertion 메트릭에서 화이트박스 방법인 GradCAM보다 우수한 성능[1]
2. **아키텍처 독립성**: VGG, ResNet 등 다양한 아키텍처에 동일하게 적용 가능[1]
3. **태스크 확장성**: 이미지 분류뿐만 아니라 이미지 캡셔닝으로도 확장 가능[1]
4. **객관적 평가**: 인간 주석에 의존하지 않는 자동 평가 메트릭 제안[1]

## 한계점

1. **높은 계산 비용**: N번의 forward pass가 필요하여 계산 집약적[1]
2. **Monte Carlo 노이즈**: 샘플링 근사로 인한 노이즈 발생 가능성[1]
3. **객체 크기 민감성**: 다양한 크기의 객체에 대한 일관성 부족 가능[1]
4. **마스크 수 트레이드오프**: 성능과 계산 비용 간의 균형점 필요[1]

## 일반화 성능 향상 가능성

### 뛰어난 일반화 능력

RISE의 일반화 성능은 다음과 같은 측면에서 탁월합니다[1]:

1. **아키텍처 독립성**: VGG, ResNet 등 다양한 CNN 아키텍처에서 일관된 성능 향상
2. **태스크 확장성**: 이미지 분류에서 이미지 캡셔닝으로의 성공적 확장
3. **데이터셋 일반화**: ImageNet, PASCAL VOC, MS-COCO 등 다양한 데이터셋에서 검증된 성능
4. **평가 메트릭 일관성**: 인간 기반 평가와 자동 인과적 평가 모두에서 우수한 결과

### 일반화를 가능하게 하는 핵심 요인

1. **모델 불가지론적 접근**: 내부 구조에 의존하지 않는 순수 입출력 기반 방법[1]
2. **확률론적 강건성**: Monte Carlo 샘플링을 통한 안정적 추정[1]
3. **이론적 기반**: 조건부 기댓값에 기반한 수학적으로 잘 정립된 프레임워크[1]
4. **포괄적 탐색**: 다양한 마스크 패턴을 통한 입력 공간의 체계적 탐색[1]

## 향후 연구에 미치는 영향

### 연구 분야에 미치는 파급효과

1. **패러다임 전환**: 설명 가능한 AI(XAI) 분야에서 블랙박스 접근법의 새로운 표준 제시[1]
2. **방법론 표준화**: 모델에 구애받지 않는 설명 방법론의 기준점 역할[1]
3. **평가 메트릭 혁신**: 인과적 평가의 중요성을 부각시키며 객관적 평가 체계 구축[1]
4. **확장성 증명**: 다양한 도메인과 태스크로의 확장 가능성 입증[1]

### 실용적 응용 분야

1. **의료 AI**: 진단 시스템의 신뢰성과 투명성 향상을 통한 임상 도입 가속화
2. **자율주행**: 주행 결정의 근거 제공으로 안전성 검증 체계 구축
3. **금융 AI**: 대출 승인, 투자 결정 등의 투명성 확보로 규제 요구사항 충족
4. **법적 규제**: EU AI Act 등 AI 규제 요구사항에 대응하는 설명 도구로 활용

## 향후 연구 시 고려사항

1. **효율성 최적화**: 계산 비용과 설명 품질 간의 최적 균형점 탐색이 필요하며, 지능적 마스크 샘플링 전략 개발이 중요합니다.

2. **도메인 특화**: 각 응용 분야의 특성을 고려한 마스크 생성 전략과 평가 메트릭 개발이 요구됩니다.

3. **확장성 확보**: 대규모 모델과 고해상도 이미지에 대한 실시간 처리 능력 개발이 필요합니다.

4. **강건성 확보**: adversarial example이나 분포 변화에 대한 설명의 안정성 보장 방법 연구가 중요합니다.

5. **다중 모달리티**: 비디오, 텍스트, 오디오 등 다양한 입력 형태로의 확장 연구가 향후 주요 과제입니다.

RISE는 블랙박스 모델 설명 분야의 획기적 발전을 이루었으며, 향후 설명 가능한 AI 연구의 방향성을 제시하는 중요한 이정표가 될 것으로 평가됩니다[1].
