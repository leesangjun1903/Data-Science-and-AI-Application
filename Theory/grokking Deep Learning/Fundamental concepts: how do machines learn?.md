# Fundamental concepts: how do machines learn?
## What is deep learning?
Deep learning is a subset of methods for machine learning.

딥러닝은 기계 학습의 하위 집합으로, 학습할 수 있는 기계를 연구하고 개발하는 데 전념하는 분야입니다(때로는 일반 인공지능을 얻는 것을 목표로 하기도 합니다).  
산업에서는 컴퓨터 비전(이미지), 자연어 처리(텍스트), 자동 음성 인식(오디오) 등 다양한 분야에서 실제 작업을 해결하는 데 딥러닝이 사용됩니다. 

머신 러닝은 컴퓨터 과학의 하위 분야로, 기계가 명시적으로 프로그래밍되지 않은 작업을 수행하는 방법을 학습합니다. 요컨대, 기계는 패턴을 관찰하고 직접적이거나 간접적인 방식으로 모방하려고 시도합니다.

저는 직접 모방과 간접 모방을 두 가지 주요 유형의 기계 학습과 병행하여 언급합니다: 지도 학습과 비지도 학습입니다.  
지도 학습은 두 데이터셋 간의 패턴을 직접 모방하는 것입니다. 항상 입력 데이터셋을 가져와서 출력 데이터셋으로 변환하려고 시도합니다. 

이 모든 작업은 지도 학습된 머신 러닝 작업입니다. 모든 경우에 머신 러닝 알고리즘은 한 데이터 세트를 사용하여 다른 데이터 세트를 예측할 수 있는 방식으로 두 데이터 세트 간의 패턴을 모방하려고 시도합니다.

## Supervised machine learning
Supervised learning transforms datasets.

지도 학습은 하나의 데이터 세트를 다른 데이터 세트로 변환하는 방법입니다. 

지도 학습은 응용 인공지능(좁은 인공지능이라고도 함)의 핵심입니다.  
알고 있는 것을 입력으로 받아 빠르게 알고자 하는 것으로 변환하는 데 유용합니다.  
이를 통해 지도 학습 알고리즘은 인간의 지능과 능력을 끝없이 확장할 수 있습니다.  
머신 러닝을 사용하는 대부분의 작업은 일종의 지도 분류기 (**supervised classifier**) 를 훈련시키는 것으로 이어집니다.  
비지도 학습조차도 일반적으로 정확한 지도 학습 알고리즘 개발을 돕기 위해 이루어집니다.

## Unsupervised machine learning
Unsupervised learning groups your data.

비지도 학습은 지도 학습과 공통적인 속성을 공유합니다: 하나의 데이터 세트를 다른 데이터 세트로 변환합니다.  
하지만 변환되는 데이터 세트는 이전에 알려져 있거나 이해되지 않았습니다. 지도 학습과 달리 모델을 복제하려는 "정답"은 없습니다.  
비지도 학습 알고리즘에게 "이 데이터에서 패턴을 찾아서 알려달라"고 하면 됩니다.

## Parametric vs. nonparametric learning
Oversimplified: Trial-and-error learning vs. counting and probability

이제 동일한 머신러닝 알고리즘을 매개변수 학습과 비모수 학습의 두 그룹으로 나누는 또 다른 방법에 대해 설명하겠습니다. 

이전 감독 섹션에서는 학습 중인 패턴의 유형에 대해 다룬 반면, 파라미터는 학습이 저장되는 방식과 종종 학습 방법에 관한 것입니다.

매개변수 모델은 고정된 수의 매개변수를 가지는 반면, 비모수 모델의 매개변수 수는 무한합니다(데이터에 의해 결정됨).

## Supervised parametric learning
Oversimplified: Trial-and-error learning using knobs

지도형 파라메트릭 학습 기계는 고정된 수의 노브(파라메트릭 부분)를 가진 기계로, 노브를 돌려 학습이 이루어집니다. 입력 데이터는 노브의 각도에 따라 처리되고 예측으로 변환됩니다.

지도형 파라메트릭 학습을 세 단계로 세분화해 보겠습니다.

### Step 1: Predict
앞서 언급했듯이 첫 번째 단계는 스포츠 통계를 수집하여 기계를 통해 전송하고 레드삭스가 우승할 확률에 대한 예측을 하는 것입니다.

### Step 2: Compare to the truth pattern
이 단계는 모델이 0%를 예측했다면 다가오는 팀의 패배를 완벽하게 예측했을 것임을 인식합니다. 기계가 정확하기를 원하므로 3단계로 이어집니다.

### Step 3: Learn the pattern
이 단계에서는 모델이 예측 시점에 얼마나 놓쳤는지(98%)와 입력 데이터(스포츠 통계)를 모두 연구하여 노브를 조정합니다. 그런 다음 이 단계에서는 노브를 돌려 입력 데이터를 고려하여 더 정확한 예측을 수행합니다. 


## Unsupervised parametric learning
비지도 학습은 데이터를 그룹화하는 것이라는 점을 기억하세요. 비지도 파라메트릭 학습은 데이터를 그룹화하기 위해 노브를 사용합니다.  
하지만 이 경우 일반적으로 각 그룹에 대해 여러 개의 노브가 있으며, 각 노브는 입력 데이터의 친화력을 해당 그룹에 매핑합니다(예외와 뉘앙스를 제외하고는 고수준 설명입니다).

각 그룹의 기계는 입력 데이터를 0에서 1 사이의 숫자로 변환하여 입력 데이터가 해당 그룹의 구성원일 확률을 알려줍니다.  
이러한 모델의 학습 방식과 결과 속성에는 다양한 종류가 있지만, 높은 수준에서는 매개변수를 조정하여 입력 데이터를 구독 그룹으로 변환합니다.

## Nonparametric learning
Oversimplified: Counting-based methods

비모수 학습은 매개변수의 수가 미리 정의된 대신 데이터를 기반으로 하는 알고리즘의 한 종류입니다.  
이는 일반적으로 어떤 방식으로든 계산하는 방법에 적합하므로 데이터 내에서 계산되는 항목의 수에 따라 매개변수의 수를 증가시킵니다.

대부분의 매개변수 모델은 여전히 데이터의 클래스 수에 따라 일종의 입력을 받아야 합니다.  
따라서 매개변수 알고리즘과 비모수 알고리즘 사이에는 회색 영역이 있음을 알 수 있습니다.  
심지어 매개변수 알고리즘도 명시적으로 패턴을 세지 않더라도 데이터의 클래스 수에 따라 어느 정도 영향을 받습니다.
