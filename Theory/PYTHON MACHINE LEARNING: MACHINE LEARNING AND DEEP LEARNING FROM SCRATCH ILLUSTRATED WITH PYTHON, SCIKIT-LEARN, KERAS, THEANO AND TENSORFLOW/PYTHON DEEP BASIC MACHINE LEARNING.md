# PYTHON DEEP BASIC MACHINE LEARNING
## 2.1 RELATING TRADITIONAL MACHINE LEARNING AND DEEP LEARNING
전통적인 머신 러닝 모델에서 직면하는 주요 과제 중 하나는 특징 추출이라는 프로세스입니다. 프로그래머는 주의해야 할 특징을 지정하고 컴퓨터에게 알려줍니다. 이러한 특징은 의사 결정을 내리는 데 도움이 됩니다. 

여러 Layer의 표현을 학습할 수 있는 딥러닝은 자동 특징 추출에 도움이 된 몇 안 되는 방법 중 하나입니다. 하위 layer는 프로그래머의 안내 없이 자동 특징 추출을 수행합니다.
