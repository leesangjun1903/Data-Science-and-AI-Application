# Random Erasing Data Augmentation

## 1. 핵심 주장 및 주요 기여  
**핵심 주장**  
Random Erasing은 학습 중 이미지에 임의의 직사각형 영역을 선택해 픽셀을 랜덤 값으로 대체함으로써 다양한 수준의 가려짐(occlusion)을 인위적으로 생성하고, 이를 통해 CNN 모델의 과적합을 억제하며 가려짐에 대한 강인한 일반화 성능을 획득할 수 있음을 제안한다.[1]

**주요 기여**  
1. 파라미터 추가 없이 손쉽게 적용 가능한 경량 데이터 증강 기법 제안  
2. 기존의 무작위 자르기(random cropping), 뒤집기(random flipping) 등과 상호 보완적이며, 이미지 분류·객체 검출·Person re-ID 전반에 걸쳐 일관된 성능 향상을 확인  
3. 학습 시 부분 가려짐을 학습 데이터에 인위적으로 추가함으로써 테스트 시 실제 가려짐 상황에도 견고한 성능 유지 검증  

## 2. 해결하고자 하는 문제  
CNN은 훈련 데이터상의 전경 객체가 완전하게 보이는 상황에 과적합(overfitting)되어, 일부가 가려진 테스트 이미지에서는 성능이 급락하는 문제가 있다. 자연스럽게 발생하는 가려짐 이미지를 수집하기 어렵고 비용이 높으므로, 학습 과정에서 인위적 가려짐을 생성해 일반화 능력을 향상시키고자 한다.[1]

## 3. 제안 방법  
### 3.1 수식 및 절차  
입력 이미지 $$I$$ (가로 $$W$$, 세로 $$H$$)에 대해 확률 $$p$$로 다음 과정을 수행한다:[1]
1. 직사각형 영역 넓이 $$S_e = U(s_l, s_h)\cdot S$$ ($$S=W\cdot H$$)  
2. 종횡비 $$r_e = U(r_1, r_2)$$  
3. 직사각형 높이 $$H_e = \sqrt{S_e \cdot r_e},\;W_e = \sqrt{S_e / r_e}$$  
4. 영역 좌표 $$(x_e, y_e)$$를 랜덤 선정, $$x_e+W_e\le W,\;y_e+H_e\le H$$ 여부 확인  
5. 해당 영역 픽셀을 $$$$ 범위 랜덤 값(또는 ImageNet 평균 픽셀값)으로 치환

이 과정을 반복해 학습 배치 내 이미지들이 다양한 가려짐 수준을 갖도록 생성한다.  

### 3.2 모델 구조에의 통합  
- **이미지 분류·Person re-ID**: 전체 이미지에 대하여 Random Erasing 적용  
- **객체 검출**:  
  - Image-aware RE (IRE): 전체 이미지  
  - Object-aware RE (ORE): 각 바운딩 박스 내부  
  - I+ORE: 전체 이미지 및 객체 영역 모두 적용  

## 4. 성능 향상 및 한계  
### 4.1 성능 향상  
- **CIFAR-10**: WRN-28-10 기준 오류율 3.80% → 3.08% 달성[1]
- **VOC2007 객체 검출**: Fast R-CNN mAP 69.1% → I+ORE 적용 시 71.5%[1]
- **Market-1501 Person re-ID**: SVDNet Rank-1 84.41% → 87.08% 향상[1]

특히 이미지 분류에서 가려짐 비율이 50%일 때도 오류율이 75%→56%로 완만하게 상승해 가려짐 강인성이 크게 개선됨을 보였다.[1]

### 4.2 한계 및 고려사항  
- 과도한 가려짐(예: $$s_h$$가 너무 큼) 시 유용 정보가 너무 많이 소실되어 성능 감소 가능  
- 최적의 하이퍼파라미터($$p, s_l, s_h, r_1, r_2$$) 탐색 필요  
- 자연 환경의 복잡한 가려짐(투명 물체, 다양한 배경) 재현에는 한계  

## 5. 향후 연구 방향 및 고려할 점  
Random Erasing은 *가려짐 강인성*을 확보하는 범용적 증강 기법으로, 향후 다음 연구에 기여할 수 있다.  
1. **적응형 가려짐 생성**: 이미지 콘텐츠·객체 위치에 기반한 동적 영역 및 형태 선택  
2. **다중 모달 증강**: 비전 이외 센서(예: 라이다) 데이터에도 유사 개념 확장  
3. **복합 증강 기법 결합**: CutMix, MixUp 등과 통합해 더욱 풍부한 데이터 다양성 확보  
4. **실제 어플리케이션 검증**: 자율주행, 의료 영상 등 실제 가려짐 시나리오 적용 및 평가  

위 고려사항을 반영하면, Random Erasing의 일반화 성능 향상 가능성을 더욱 극대화할 수 있을 것이다.
